# Warm-mode hero illustrations (runtime-served)

This folder is served by Vite at the URL `/assets/warm/illustrations/`. Drop
artwork files here so `<WarmHero />` picks them up automatically; without
artwork, the component renders an abstract gradient placeholder.

See `src/assets/warm/illustrations/README.md` for full sourcing guidance.

## Expected files
- `mind-wellbeing.png` — Browser/Web Access screen
- `activities-landscape.png` — Activities landing
- `daily-care-paper.png` — Medical → Daily Care landing
- `design-home-blueprint.png` — Design Home landing

Recommended size: 1600×700 (16:7) or larger, warm palette
(cream/sage/gold tones).
