# Warm-mode hero illustrations

Drop PNG or JPG files here to replace the abstract gradient placeholders shown
by `<WarmHero />` in warm theme. Required dimensions: 1600×700 (16:7) or larger,
optimized for warm palette (cream/sage/gold tones).

## Expected files
- `mind-wellbeing.png` — used on the Browser/Web Access screen
- `activities-landscape.png` — used on the Activities landing
- `daily-care-paper.png` — used on Medical → Daily Care landing
- `design-home-blueprint.png` — used on the Design Home landing

## Sourcing
Options for licensed illustrations:
- Unsplash (free, attribution): warm-toned landscape / botanical / paper textures
- iStock / Adobe Stock (paid, no attribution)
- Commission an illustrator on Upwork / Fiverr / 99designs (~$50–$200 per piece)
- AI generation (Midjourney / DALL·E) — verify license terms for commercial use

Until artwork is dropped here, `WarmHero` automatically renders a beautiful
abstract gradient that matches the warm palette. The app is fully functional
without illustrations — they are progressive enhancement only.
